Created by Nimish Patil

About Tipworthy

Tipworthy is a dynamic content recommendation system designed for users of ChangeTip. It deploys a topic modeling algorithm on content users have previously tipped to determine their preferences and makes recommendations for new incoming content.

Try it out!

You can try out the app by picking one of the anonymous users below.

User 1 User 2 User 3 User 4 User 5

Each user is an actual ChangeTip user that has given at least one tip on Reddit for some post they appreciated. Using Latent Dirichlet Allocation, every tipped post was assigned a set of topic distributions. Based on the amount tipped, the Tipworthy algorithm determined the user's preference for each topic.
